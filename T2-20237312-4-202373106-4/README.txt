Consideraciones antes de ejecutar:
solo pueden haber 2 participantes en un articulo, también un revisor
no puede subir articulos, solo los autores pueden hacerlo.
para asignar mas de un topico hay que mantener apretado Ctrl y hacer clic
en los topicos deseados (de igual manera la pagina avisa como se debe hacer).
la fecha en la que se sube el articulo debe ser ingresado por el usuario
uno debe ingresar el participante independiente si ya este o no ingresado en la base de datos
en la pestaña de busqueda, solo se puede aplicar el filtro para un topico a la vez

0) iniciar la aplicacion xampp, y activar el xampp y mysql
1) Ir a la carpeta ../xampp/htdocs y crear una carpeta llamada Tarea2BD y meter todos los archivos descargados del aula
2) ingresar a (http://localhost/phpmyadmin/) y crear la base de datos que se debe llamar congresodb
3) hay que crear la tabla desde la vista sq1 y copiar y pegar el contenido del archivo 01_Creacion_TablaNormalizada
4) después se deben ingresar los datos de la población que fueron adjuntados en el archivo Insert_Completo_50_CORREGIDO en la pestaña
sql y después apretar el boton continuar para que se ingresen todos los datos
